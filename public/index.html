How to use in OBS:

  1. Open OBS Studio
  2. Add a new source → Browser Source
  3. Paste the widget URL above
  4. Set width: 800px and height: 600px
  5. Check "Shutdown source when not visible"
  6. Don’t remove anything after # in the URL
  7. Click OK and position as needed
